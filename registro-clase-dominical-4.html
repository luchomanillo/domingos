📊 Conexión a Google Sheets

Paso a paso

1
Abrí Google Sheets y creá una hoja nueva. Nombrála Registro Dominical.
2
En la fila 1, escribí estos encabezados exactamente en las celdas A1 a G1:
Fecha | Hora | Nombre Niño | Fecha Nac | Salón | Nombre Adulto | Parentesco | Teléfono | N° Control
3
Andá a Extensiones → Apps Script. Borrá lo que haya y pegá el código que está más abajo.
4
Hacé clic en Implementar → Nueva implementación → Aplicación web. Configurá: Ejecutar como Yo, Acceso para Cualquier usuario. Copiá la URL que te da.
5
Pegá esa URL en el campo de abajo y guardá.
📋 Código para Apps Script: function doPost(e) { try { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var p = e.parameter; sheet.appendRow([ p.fecha, p.hora, p.nombreNino, p.fechaNac, p.salon, p.nombreAdulto, p.parentesco, p.telefonoAdulto, p.numControl ]); return ContentService .createTextOutput(JSON.stringify({ok: true})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({ok: false, error: err.toString()})) .setMimeType(ContentService.MimeType.JSON); } } function doGet(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var rows = sheet.getDataRange().getValues(); var headers = rows[0]; var data = []; for (var i = 1; i < rows.length; i++) { var obj = {}; for (var j = 0; j < headers.length; j++) { obj[headers[j]] = rows[i][j]; } data.push(obj); } return ContentService .createTextOutput(JSON.stringify(data)) .setMimeType(ContentService.MimeType.JSON); }
